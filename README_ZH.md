# 异步 I/O 调度器 —— uasyncio

中文 | [English](README.md)

## 概述

uasyncio 是用来编写 **并发** 代码的库，使用 **async/await** 语法。uasyncio 被用作多个提供高性能异步框架的基础，包括网络和网站服务，数据库连接库，分布式任务队列等等。uasyncio 往往是构建 IO 密集型和高层级 **结构化** 网络代码的最佳选择。
